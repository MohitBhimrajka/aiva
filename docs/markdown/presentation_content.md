### **Slide 6: Technical Architecture: The AIVA Engine**
**Speaker:** Mohit (The Competent Architect)
**Time:** ~75 seconds

**(Visual Guidance)**
*   **Title:** The AIVA Engine: A Modern, Scalable Architecture
*   A polished, high-level flow diagram. Use logos for key technologies (Next.js, FastAPI, GCP, HeyGen, Gemini).
*   The diagram should clearly show the flow:
    *   **User/Browser (Next.js)** on the left, showing Mic/Camera input.
    *   An arrow goes to the **Backend (FastAPI on GCP Cloud Run)** in the center.
    *   The Backend connects to three external services:
        *   **Google Speech-to-Text** (for audio transcription)
        *   **Vertex AI / Gemini** (for content analysis)
        *   **HeyGen API** (to generate avatar speech)
    *   The Backend also has a two-way connection to the **Database (PostgreSQL on Cloud SQL)**.
*   **Key:** Keep it high-level. Avoid showing every single API endpoint. The goal is to show the *system*, not the implementation details.

**(Speaker Script)**
"Powering this seamless experience is a modern, scalable technical architecture. Let's walk through the flow of a single answer.

It starts at the **Frontend**, built with Next.js. Here, we capture the user's audio and video, and render the HeyGen streaming avatar. For facial analysis, we use MediaPipe.js to process video *directly in the browser*, ensuring user privacy by only sending metadata, not the video feed itself.

This data is sent to our **Backend**, a FastAPI application running on GCP Cloud Run for serverless scalability. This backend is the central orchestrator.

It sends the user's audio to **Google's Speech-to-Text API** for highly accurate transcription. That text, along with the facial metadata, is then sent to **Google's Gemini model** for deep analysis and scoring.

Simultaneously, the next question is sent to the **HeyGen API**, which generates the avatar's response streamed back to the user. All session data, user history, and feedback reports are stored securely in a **PostgreSQL** database. This entire stack is chosen for scalability, reliability, and its powerful AI capabilities."
